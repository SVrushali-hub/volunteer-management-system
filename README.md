# Volunteer Management System

A full-stack Volunteer Management System developed for **NayePankh Foundation** to streamline volunteer registration, approval workflows, reporting, and administration.

## Features

* Volunteer Registration
* Admin Authentication
* Volunteer Approval & Rejection
* Volunteer Search
* Status Tracking
* Reports Dashboard
* CSV Export
* Responsive UI
* MongoDB Database Integration

## Tech Stack

### Frontend

* React.js
* React Router DOM
* CSS
* Framer Motion

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

## Installation

### Frontend

cd client
npm install
npm run dev

### Backend

cd server
npm install
npm start
